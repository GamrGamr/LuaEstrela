What it does

Shut down on time.
Stay in control.

Halvynox helps a Windows computer follow a reliable daily shutdown schedule. Before shutting down, it displays a live countdown and lets the user delay the action while configured attempts remain. Because it initiates a real shutdown, save open work before the scheduled time.

✓ Schedule one daily shutdown time
✓ See a warning popup with a live countdown
✓ Configure a limited number of delay attempts
✓ Keep delay tracking across restarts and midnight
✓ Protect settings and tray exit with a password
✓ Start automatically with Windows
✓ Minimize quietly to the system tray
✓ Choose between light and dark themes
