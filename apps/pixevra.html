What it does

Shape every image.
Keep it private.

Pixevra brings common image tasks into one portable desktop app. Convert between popular formats, resize and crop precisely, build Windows icons, trim empty space, and clean up unwanted backgrounds.

✓ Convert PNG, JPG, WEBP, BMP, TIFF, GIF, and ICO
✓ Resize while preserving aspect ratio
✓ Crop manually or trim transparent space
✓ Create multi-size Windows ICO files
✓ Remove white and edge-connected backgrounds
✓ Batch-convert selected images
✓ Polish transparency edges and preview output
✓ Dark and light themes with saved settings
